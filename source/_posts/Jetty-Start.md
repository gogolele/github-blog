### 1. start

jetty目前版本为9，我们目前使用的是稳定版本 6。1.26.

下载地址：
```
http://download.eclipse.org/jetty/stable-9/dist/
```
一些常见的启动命令
```
java -jar start.jar --auto-to-install=http,deploy
java -jar start.jar --dry-run
java -jar start.jar --add-to-startd=http,deploy
```

1.7_65后，G1GC很不错。
